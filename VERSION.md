# 版本資訊

## v1.0.0 (2025-12-13)

### 🎉 首次發布

#### 功能特色

- ✅ 三階段 AI 教材生成系統
  - 大綱生成（結構化規劃）
  - 教材生成（詳細內容）
  - 題目生成（練習與解析）

- ✅ 多科目支援
  - 數學、自然科學、理化、生物、物理、化學
  - 國文、英文、歷史、地理

- ✅ 年級客製化
  - 涵蓋小一到高三
  - AI 自動調整內容難度

- ✅ 歷史記錄管理
  - 自動儲存所有生成內容
  - 搜尋與篩選功能
  - 重新查看過往教材

- ✅ 現代化 UI
  - 響應式設計
  - Tailwind CSS 美化
  - 清晰的步驟指示器

#### 技術實作

**後端**
- FastAPI 框架
- SQLite 資料庫
- Google Gemini API 整合
- Chain of Thought 提示工程

**前端**
- React 18
- Vite 建構工具
- Tailwind CSS
- Axios HTTP 客戶端

#### 文件

- 完整的 README.md
- 快速安裝指南 (SETUP_GUIDE.md)
- 快速開始文件 (QUICKSTART.md)
- 專案結構說明 (PROJECT_STRUCTURE.md)
- 後端 API 文件
- 前端開發指南

#### 開發工具

- Windows 啟動腳本
- Git 忽略設定
- 環境變數範本
