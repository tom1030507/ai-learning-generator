## 📖 使用者指南

### 第一次使用（預估 5 分鐘）

1. **安裝環境**（2 分鐘）
   - 確認 Python 和 Node.js 已安裝
   - 取得 Gemini API Key

2. **設定專案**（2 分鐘）
   - 執行後端啟動腳本
   - 設定 .env 檔案
   - 執行前端啟動腳本

3. **開始使用**（1 分鐘）
   - 開啟瀏覽器
   - 輸入教材資訊
   - 開始生成

### 生成教材（預估 30-60 秒）

1. 大綱生成：10-15 秒
2. 教材生成：10-20 秒
3. 題目生成：10-25 秒

## 🐛 已知限制

1. **API 配額限制**：Gemini 免費版有使用上限
2. **單一資料庫**：SQLite 適合小型應用，大規模需升級
3. **無認證系統**：目前任何人都可使用（可新增）
4. **無多語言**：目前僅支援繁體中文
5. **圖片功能**：暫不支援生成或插入圖片
